# Базіка - Telegram Агент

Workspace для агента Базіка (Telegram бот з характером).

## Структура

```
~/.openclaw/
├── agents/
│   └── bazika/            # Базіка 
└── workspace-bazika/      # Workspace для Базіки (ЦЕЙ КАТАЛОГ)
    ├── SOUL.md            # Характер Базіки
    ├── IDENTITY.md        # Хто такий Базіка
    ├── USER.md            # Опис користувачів
    └── README.md          # Цей файл
```


## Характер

Базіка — український бот з характером:
- Сленгова українська мова
- Під'їбає, але допомагає
- Матюкається коли доречно
- Не терпить маячні
- Може послати якщо питання дурне

## Редагування поведінки

Щоб змінити характер Базіки, редагуй файли в цьому каталозі:
- `SOUL.md` - основний характер і стиль
- `IDENTITY.md` - хто такий Базіка
- `USER.md` - для кого він працює

Зміни підхоплюються автоматично (перезапуск не потрібен).

## Перевірка

```bash
# Список агентів
openclaw agents list

# Routing bindings
openclaw agents bindings

# Статус каналів
openclaw channels status --probe
```

## Логи

```bash
# Всі логи
openclaw logs --follow

# Тільки для Bazika (якщо можна фільтрувати)
openclaw logs --follow | grep bazika
```
